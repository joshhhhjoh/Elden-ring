Overview (What you’re doing)

Run Elden Ring offline with EAC disabled, attach Cheat Engine to eldenring.exe, spawn runes/items, and keep a clean save for online play.

Save path: C:\Users\<You>\AppData\Roaming\EldenRing\<SteamID>\ER0000.sl2

Prep (once)

1
Back up your save (ER0000.sl2) to a safe folder.
2
Set Steam → Go Offline and restart Steam.

Disable Easy Anti‑Cheat

1
Open ...\Steam\steamapps\common\ELDEN RING\Game\
2
Rename start_protected_game.exe → start_protected_game.exe.bak
3
Copy eldenring.exe and rename that copy to start_protected_game.exe

Launching now uses your copy (no EAC) so CE can attach safely.

Launch & Attach Cheat Engine

1
Launch via your new start_protected_game.exe.
2
Open Cheat Engine → attach to eldenring.exe.
3
Load your table (ItemGib / AddSoul / TGA) and enable needed scripts.

Using CE Tables + Online‑Safe Runes (Offline Only)

What a CE Table (.CT) is
  • A .CT file is a Cheat Engine script bundle (tables + scripts + hotkeys).
  • Different authors: Hexinton, Marcus101RR, The Grand Archives (TGA), etc.
  • Tables often require a specific game version/DLC. Pick one that matches your build.
How to load and enable scripts
  1. Launch Elden Ring offline with EAC disabled.
  2. Open Cheat Engine → File → Open → choose your .CT table.
  3. Click the PC icon in CE and attach to eldenring.exe.
  4. In the table list, tick the master/enable script (AOB/enable) until it says Activated.
  5. Tick the specific modules you need: ItemGib, AddSoul, etc.
  6. Most tables show hotkeys in the Description column; right‑click to change if you want.
Give yourself runes (offline) — online‑safe workflow
  1. Back up your clean save (ER0000.sl2) somewhere safe.
  2. With Steam offline and EAC disabled, enable AddSoul in CE.
  3. Enter a rune amount (e.g., 500000) and run the script while in‑game.
  4. Spend the runes offline (leveling, purchases, etc.).
  5. When finished, restore your clean save and put back the original start_protected_game.exe before going online.

Important: Never go online with a save that was edited or created while EAC was disabled. Always return to your clean save first.

Quick troubleshooting
  • Script won’t activate: Attach after the main menu, or use a table version that matches your patch.
  • CE can’t see the process: Verify you launched the EAC‑less executable.
  • No effect in game: Re‑enable the script, re‑enter values, and try a small in‑game action to trigger a refresh.

Duplicate Loop (optional)

  1. Drop spawned items on the ground.
  2. Quit the game.
  3. Restore your backup over ER0000.sl2.
  4. Launch offline again → pick up → repeat.

Pro tip: Keep CLEAN_SAVE (for online) and WORKING_SAVE (for offline edits).

Clean Online Return

  1. Delete the copied start_protected_game.exe.
  2. Rename your .bak back to start_protected_game.exe.
  3. Restore your clean, unmodded save.
  4. Set Steam online and launch normally.

Troubleshooting

Game won’t launch offline
  • Recheck Steam’s Offline mode (restart Steam).
  • Verify the EAC rename/copy steps.
CE can’t attach
  • Confirm you launched the EAC‑less executable.
  • Attach to the correct process (eldenring.exe).
